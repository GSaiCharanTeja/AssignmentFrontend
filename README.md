# Task Management Frontend

A modern Task Management Web Application built using React, Vite, Tailwind CSS, and Axios.

## Features
- User Authentication
- JWT Token Handling
- Create / Update / Delete Tasks
- Responsive UI
- Protected Dashboard
- API Integration with Spring Boot Backend
- Deployed on Vercel

## Tech Stack
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM
